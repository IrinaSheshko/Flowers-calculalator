# 💐 Flower Shop Bouquet Calculator

Интерактивное приложение для цветочного магазина, которое позволяет пользователям собрать индивидуальный букет и мгновенно рассчитать его стоимость.

## 🚀 [Live Demo](https://IrinaSheshko.github.io/Flower-calculator/)

---

### 🌟 Функционал:
* **Динамический расчет:** Калькулятор учитывает стоимость цветов, их количество, декоративную зелень и тип упаковки.
* **Валидация форм:** Использование атрибутов `required` и `min` для предотвращения некорректного ввода.
* **UX/UI:** Удобные выпадающие списки (select) с иконками эмодзи для наглядности.
* **Адаптивность:** Форма подстраивается под экраны мобильных устройств.

---

### 🛠 Стек технологий:
* **JavaScript (Vanilla):** Работа с DOM-деревом, событиями `click` и математическими операциями.
* **HTML5:** Семантические формы, элементы навигации и футер.
* **CSS3:** Flexbox для центрирования формы и стилизации интерфейса.

---
